### Customizing Theme
You can customize the theme by overriding the CSS variables.
```css
:root {
  --fui-color-brand: var(--fui-color-green-700);
}

[data-theme="dark"],
[data-theme="dark"] * {
  --fui-color-brand: var(--fui-color-green-500);
}
...
```

List of all the variables and their default values (both light and dark theme) can be found in the [theme colors file]().
